- 200+ first-party connectors

- Sandboxed runtime with audit trails
- Per-project encrypted secret vault
- Multi-region deployment for GDPR
- Powered by MiMo-Series, Claude-Series, DeepSeek-Series

## Quickstart
```bash
curl -fsSL verdant-farmers-market.dev/install | sh
verdant init --model mimo-v2.5-pro
```
